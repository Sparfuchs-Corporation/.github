# Welcome to Sparfuchs Corporation 👋

**Aligned Execution and Governance for Intelligent Systems.**

Enterprises are shipping AI agents faster than they can govern them. Every LLM call is a budget leak. Every agent write-action is an audit exposure. We build the platform stack that closes that gap — so agentic AI becomes a board-defensible operating advantage, not a seven-figure experiment.

This is the open-source home of **the A.E.G.I.S. Methodology** — **A**ligned **E**xecution and **G**overnance for **I**ntelligent **S**ystems — and its supporting platform stack.

## The Platform Stack

The three tiers that operationalize the framework:

| Layer                              | Platform                       | What it does                                                                                                                                                                                                                                                                                         |
| ---------------------------------- | ------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1 — Quality Foundation**         | **Sparfuchs QA**               | 42 specialist review agents. 26 deterministic canaries. Multi-provider orchestration with script, hybrid, and LLM execution modes — so you pay the LLM tax only where intelligence is actually required.                                                                                             |
| **2 — Visibility & Control**       | **agentForge + businessForge** | Visual canvas for agent design. Fail-closed policy engine. 4-layer ABAC. Per-node cost chips. Token budgets with 80% alerts and 100% auto-pause. Graduated autonomy — read-only, then approve-then-act, then auto-approve-with-undo. Digital attestation on every release. Zero-trust release gates. |
| **3 — Enterprise AI Intelligence** | **businessForge**              | AI-vs-Software classification on every requirement, across 8 evaluation factors. Immutable versioning. Scheduled re-evaluation. Emergent problem discovery. One pane of governance over every agent and every dollar of AI spend.                                                                    |

No single competitor spans all three. Individual pieces are replicable. The combination is not.
